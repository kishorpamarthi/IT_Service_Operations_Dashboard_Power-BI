# IT Service Operations Performance Dashboard (2024)

This Power BI dashboard analyzes 5,000 IT support tickets from a simulated MSP/ITSM dataset based on real-world
ServiceNow/ITIL patterns. The report focuses on operational efficiency, service delivery performance, SLA compliance,
and workload distribution across priority levels and support teams.

## Key Metrics

- Total Tickets: 5,000
- SLA Compliance: 86.14%
- Average Resolution Time (MTTR): 20.62 hours
- Reopen Rate: 11.58%
- Escalation Rate: 21.94%

## Dashboard Highlights

- SLA performance trend tracking across months
- Ticket workload by priority, category, and assignment group
- Escalated vs non-escalated incident analysis
- Ticket volume trend analysis
- Filter panel for drill-down: customer, priority, group, category, and date
